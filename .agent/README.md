# .agent フォルダについて

このフォルダは、エージェント作業用のドキュメント置き場です。

## 構成

- `plans/`
- 実行中のプランを格納するフォルダです。
- 現在は `STEP1.md` があり、自由形式テキストをLLMで解釈してタスクを保存・参照する基盤（Google Drive上の `tasks.md` 利用）の方針、スコープ、受け入れ条件の概要が記載されています。
- `done/`
- 完了したプランの Markdown ファイルを格納するフォルダです。

## 運用ルール

- プランが完了したら、対象の Markdown ファイルを `.agent/plans/` から `.agent/done/` へ移動します。
- この移動作業は常にエージェントが自分で実行します。
